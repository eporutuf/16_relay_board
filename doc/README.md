**16_relay_board**</br>
**Communication:**</br>
***UDP Message | Details-</br>***
 FF0101	| Relay 1 ON command (ASCII encoding)-FF0100	Relay 1 OFF command (ASCII encoding)</br>
 ...</br>
FF1600 |	Relay 16 OFF command (ASCII encoding)</br>
FF1601 | Relay 16 ON command (ASCII encoding)</br>
FFE000 |	All relays OFF command (ASCII encoding)</br>
FFE001 |	All relays ON command (ASCII encoding)</br>
FFT000 |	Get temperature value (only available with "sensors" board option)</br>
FFC000 |	Get current value (only available with sensors relay board option)</br>

##**Features:**</br>
STM32F411RE Microprocessor, 32-Bit ARM Cortex</br>
W5500 Ethernet module</br>
16 Relays Output</br>
1x Jack connector</br>
1x DS18B20 JACK wire 10meters</br>
ACS722 current sensors</br>
2x16 Terminals fast connector 24 AWG</br>
SMD DIP SWITCH 6
